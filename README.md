# digraph

Digraph module provides several functions for working with
[directed graph](http://en.wikipedia.org/wiki/Directed_graph)s (digraphs)
in JavaScript.

## node.js
```js
var digraph = require('digraph');
var g = digraph();
```

## browser

Use browserify.

## API

### g.add.node(<node>)

### g.add.edge.from(<node>).to(<node>)
